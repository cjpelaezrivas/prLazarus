Plantilla Java para lectura y escritura de ficheros de texto

jueves, 7 de agosto de 2014
Tiempo de lectura: 1 minutos
Etiquetas: Eficiencia Java Tutoriales Utilidades

Muy buenas a todos,
hoy os traigo una pequeña aplicación que creo que puede ser muy útil para muchos. De echo, la subo aquí para tenerla a mano siempre que la necesite.

Se trata simplemente de una plantilla escrita en Java para la lectura y escritura de ficheros en texto plano. Esto puede llegar a ser muy útil porque se trata un trozo de código común en muchas aplicaciones sencillas, como por ejemplo limpieza de texto o extracción de información de un archivo HTML.

El código es muy sencillo y consta de dos partes bien diferenciadas, por lo que solo habrá que completar una de las etiquetas TODO de los bucles que hay. Dicha elección es por parte del usuario al elegir la fuente de entrada: por consola, o desde un archivo.

A continuación os pongo el código para la descarga por si os interesa.

Saludos y hasta la próxima.
Lázarus Surazal.
